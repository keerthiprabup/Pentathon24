Screen Reader Access

Pune Metro website complies with World Wide Web Consortium (W3C) Web Content Accessibility Guidelines (WCAG) 2.0 level AA. This will enable people with visual impairments access the website using assistive technologies, such as screen readers. The information of the website is accessible with different screen readers.

Sr. No Screen Reader Website
1) Non Visual Desktop Access (NVDA) https://www.nvaccess.org/download
2) System Access to Go http://www.satogo.com
3) Thunder http://www.screenreader.net/index.php?pageid=2
4) Web Anywhere http://www.webanywhere.cs.washington.edu/wa.php
5) Screen Access for All (SAFA) http://www.nabdelhi.org/NAB_SAFA.htm
